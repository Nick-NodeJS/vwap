## Description

VWAP Calculator

## Tech stack
- Go
- Websocket
- Bitfinex API

## Installation


See https://go.dev/doc/install

## Build

```bash
$ go build -o vwap ./cmd
```
As a result binary file 'vwap' should be created in root directory

## Running the app
#### Double check .env file. It should have everything like env.example

```bash
$ ./vwap
```
or

```bash
$ go run ./cmd
```

## Test

```bash
$ go test ./cmd
```
